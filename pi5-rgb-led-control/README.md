# Pi5 RGB Status Link.

A Raspberry Pi 5 LED controller for Volumio that matches colour to audio resolution. 

### Features
* **Real-time Tracking:** Supports DSD and PCM bitrate monitoring.
* **System Animations:** Visual feedback for system states and the "Singularity" shutdown sequence.
* **Stability:** Includes a 3-strike network buffer to prevent jitter.
* **Customisation:** Full web UI for pin and colour settings.

High-fidelity visuals for high-fidelity audio.
